# RavolOS

RavolOS is a hobby operating system designed to learn and experiment with low-level programming. It features a custom kernel and a custom bootloader, 
with plans to switch to GRUB once the OS is more mature.

## Features

- Fully custom kernel
- Simple build process via the `start.sh` script

## Working on

- Task handling
- FAT32 FS support

## Building and Running

To compile and run use the provided bash script:

```bash
./start.sh
```
